# 🏋️ FixMyForm

**An AI-powered workout form coach that gives you instant, honest feedback on your exercise form, just from a photo.**

No more guessing. No more paying a trainer just to check your squat depth. Upload a photo mid-rep and get real feedback on what's right, what's wrong, and how to fix it.

🔗 **Live:** [fix-my-form-vert.vercel.app](https://fix-my-form-vert.vercel.app)

---

## ✨ Features

### 🏋️ Exercise Explorer
Browse 50+ exercises across 11 muscle groups. Each exercise comes with reference photos and form cues so you know exactly what correct form looks like before you even attempt it.

### 📸 AI Form Checker
Upload a photo of yourself mid-rep. The AI analyzes your form against the selected exercise, tells you what you're doing right, what needs fixing, and suggests 2-3 alternative exercises if needed.

### 🔄 Smart Swapper
Can't do a certain exercise? Missing equipment, dealing with joint pain, or just training at home? Filter by muscle group, equipment, and constraint to find the right biomechanical alternative in seconds.

### ⚡ Lifter Toolkit
A full built-in utility suite so you never have to open another tab mid-workout:
- 1-Rep Max Calculator
- Barbell Plate Calculator
- TDEE Calculator
- BMI & BMR Calculator
- Body Fat % Estimator
- Form Score Tracker
- Rest Timer

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **AI:** Groq Vision AI
- **Exercise Data:** ExerciseDB API

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/jatinnagwani/FixMyForm.git

# Navigate into the project
cd FixMyForm

# Install dependencies
npm install

# Add your environment variables
# Create a .env file and add your Groq API key + ExerciseDB API key

# Run the dev server
npm run dev
```
